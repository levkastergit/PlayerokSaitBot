# Шаг 0 — перехват трафика покупки Robux (Microsoft Store)

Цель: один раз вручную купить минимальный пак Robux в приложении Roblox (Microsoft
Store), оплатив **балансом Microsoft-аккаунта**, и записать сетевые запросы. По ним
станет видно, какие именно вызовы делает приложение, и какой из двух путей реализации
Фазы 2 возможен (воспроизведение запросов vs UI-автоматизация приложения).

> ⚠️ Делайте это на **тестовой машине/VM** с **тестовыми** аккаунтами Roblox и Microsoft.
> Это реальная покупка (тратится баланс), системное доверие к перехватывающему сертификату
> и риск флага/бана аккаунтов. Не на основном ПК и не на основных аккаунтах.

## Что понадобится
- Windows 10/11.
- Python + mitmproxy (`pip install mitmproxy`) — уже стоит, если выполнен Шаг 0 из основного README.
- **Приложение Roblox из Microsoft Store** (именно UWP-версия из Store, не клиент с roblox.com —
  только в Store-версии есть оплата «Microsoft account balance»). Установка: откройте
  **Microsoft Store → найдите Roblox → Установить** (нужен вход в Store под MS-аккаунтом).
- Microsoft-аккаунт с балансом (от подарочной карты), регион аккаунта совпадает с валютой карты.

## Шаги
1. **Сгенерировать сертификат mitmproxy** (если ещё не создан): один раз запустите `mitmdump`
   и сразу закройте (Ctrl+C). Он создаст `%USERPROFILE%\.mitmproxy\mitmproxy-ca-cert.cer`.
2. **Доверить сертификат** — в PowerShell **от администратора**:
   ```powershell
   powershell -ExecutionPolicy Bypass -File .\trust-cert.ps1
   ```
3. **Войти** в приложении Roblox под тестовым аккаунтом Roblox; в Microsoft Store — под
   MS-аккаунтом с балансом.
4. **Запустить перехват** (обычный PowerShell, в этой папке):
   ```powershell
   powershell -ExecutionPolicy Bypass -File .\start-capture.ps1
   ```
   Скрипт включит loopback-исключение для Roblox, системный прокси на mitmproxy и начнёт запись.
5. **Купить минимальный пак Robux** (например 80/400 R$) в приложении Roblox, выбрав оплату
   **«Microsoft account balance»**, и подтвердить.
6. **Остановить** перехват (Ctrl+C). Прокси вернётся в прежнее состояние.
7. **Снять доверие** к сертификату (гигиена): `trust-cert.ps1 -Remove` (от админа).

## Что прислать мне
- `robux-capture.txt` — читаемая выжимка только релевантных запросов (Microsoft + Roblox).
- `robux-capture.flows` — полный дамп (на случай, если в txt чего-то не хватит).

Ищем в записи:
- `*.microsoft.com` / `purchase.mp.microsoft.com` / `licensing*.xboxlive.com` — оплата/лицензия;
- `collections.mp.microsoft.com/.../consume` — это «Generate Store ID» → «Claim» у свизера;
- `billing.roblox.com` или `apis.roblox.com` (ищите `windows`/`xbox`/`microsoft` в пути) —
  запрос, который кладёт Robux на аккаунт Roblox.

## Если в записи нет расшифрованных тел (HTTPS не вскрылся)
Значит приложение/Microsoft **пинят сертификаты** — обычный системный прокси их не вскроет.
Тогда Фаза 2 почти наверняка пойдёт по пути **UI-автоматизации** приложения (pywinauto/FlaUI/
WinAppDriver) вместо воспроизведения запросов. Это нормальный исход — напишите мне, что тела
не вскрылись, и мы пойдём этим путём.
